# Система паттернов — индекс

Каждый паттерн — отдельный файл. Папки соответствуют **фазе главы**, в которой паттерн обычно применяется. Файлы с префиксом `00_` — служебные (этот индекс, таблица конфликтов, рекомендованные связки).

При сборке промпта для конкретной фазы главы сборщик читает только соответствующую папку. Это держит промпт компактным.

---

## 00_global — сквозные стилистические паттерны

Применяются **по всей главе**, не привязаны к конкретному месту.

| ID | Название | Источник | Что делает |
|---|---|---|---|
| `style_short_sentences` | Короткие предложения, активный залог | Thompson | Предложения 5-15 слов, активный залог, бытовые слова. |
| `style_pronunciation` | Как читать вслух | Thompson | При введении нового символа — одна строка «читается так-то». |
| `style_voice_you` | Прямое обращение «вы» | Thompson | Обращение «вы» (не «читатель», не «мы»), создание диалога. |

## 01_chapter_opening — открытие главы

Применяются в **первых 1-3 абзацах** главы. Их задача — задать тон, заслужить доверие, мотивировать читать дальше.

| ID | Название | Источник | Что делает |
|---|---|---|---|
| `open_self_deprecation` | Самоуничижение автора | Thompson (T1) | «Я тоже не понимал, постараюсь объяснить лучше». ✅ полный файл |
| `open_naming_enemy` | Называние «врага» | Thompson (T2) | «В обычных учебниках это место выглядит как наказание...» |
| `open_story_first` | Сценка раньше определения | Blitzstein (B1) | Бытовая ситуация, из которой вырастает понятие. |
| `open_manifesto` | Манифест-обещание главы | Savov (S1) | «В этой главе мы X, и обещаем не Y». |
| `open_great_minds_erred` | «И Ньютон ошибался» | Blitzstein (B8) | «Эту задачу сначала ошибочно решил даже Лейбниц». |
| `open_promise_payoff` | Анонс кульминации | Blitzstein (B2 модиф.) | «К концу главы вы увидите, что одно число знает четыре вещи». |

## Что НЕ перенесено

Некоторые паттерны исходного списка не попали в новую структуру осознанно:

- **S3 (density block)** — оставлен только как тип жанра внутри других паттернов, не отдельный паттерн.
- **S4 (humor about Monsanto)** — политический юмор Savov, не подходит нашей аудитории.
- **S5 (definitions list)** — поглощён внутри `intro_multiple_repr` и `close_what_we_learned`.
- **S6 (formal notation without delay)** — конфликтует с целевой аудиторией «с нуля», не применяется.

Это даёт **27 активных паттернов** вместо изначальных 31. Зато все 27 — рабочие, не дубликаты.
