# Суть

При первом появлении нового символа — **одна короткая строка** о том, как он читается вслух по-русски. Снимает скрытый страх «я не знаю, как это назвать», и даёт читателю присвоить понятие через звук.

# Формат

«Запись X читается как „...“».

Опционально — короткий комментарий, **зачем** важно уметь произнести: «пока вы не можете произнести формулу, вы её не можете думать».

# Когда применять

- Первое появление любого символа греческого алфавита.
- Первое появление спецзнака ($\sum$, $\int$, $\partial$, $\nabla$, $\vec{v}$, $\det$).
- Нотация с индексами при первом введении ($a_{ij}$, $x_n$).
- Запись определителя ($|M|$ vs $\det M$).

# Не применять

- В каждой следующей главе для того же символа.
- Для общеизвестных символов ($+$, $-$, $\times$, $\div$).

# Связка

Хорошо работает вместе с `intro_disarm_symbols` (разоружение значения) — произношение даёт звук, разоружение даёт смысл, вместе они полностью присваивают символ.
