# Суть

Внутри блока задач — **явная маркировка по сложности**, чтобы читатель мог калибровать свои силы и видеть прогресс.

Три уровня:
- **○ Разминка** — задачи на прямое применение формулы. 5-10 минут на каждую.
- **◐ Основа** — задачи, требующие выбора метода или короткой комбинации. 10-20 минут.
- **● На подумать** — задачи на нестандартное применение, ловушки, синтез. 20+ минут.

Маркеры (○/◐/●) ставятся **перед номером задачи** и видны сразу.

# Структура с маркировкой

```
### Разминка по теме

○ 1. Посчитать определитель матрицы (2 0; 1 3).
○ 2. Применить матрицу (1 1; 0 1) к вектору (3, 2).
◐ 3. Для какой матрицы (a b; c d) определитель равен a + d?
● 4. Доказать, что определитель кососимметричной матрицы 2×2 неотрицателен.
```

# Когда применять

- В **каждом блоке задач**.

# Когда не применять

- Если в главе всего 5-7 задач (короткая глава) — уровни не нужны.

# Распределение

Для блока из 25 задач рекомендованное распределение:
- Разминка ○ : 10-12 задач
- Основа ◐ : 8-10 задач  
- На подумать ● : 3-5 задач

# Не путать с

`tasks_strategic_vs_mixed` — это деление **по теме** (одна или несколько).
`tasks_difficulty_levels` — это деление **по сложности** (легче или труднее).

Эти оси **ортогональны**. У задачи есть и тип, и уровень. Смешанная задача может быть лёгкой (на узнавание), а тематическая — трудной (на ловушку внутри одной темы).

# Сборщик

Скрипт-генератор задач (этап 5) маркирует каждую задачу:
- по фактическому числу логических шагов в решении,
- по тому, сколько нестандартных приёмов требуется,
- по типу: применение / выбор метода / синтез.

Это автоматизируемо.
