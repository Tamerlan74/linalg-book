# Суть

Задачи в конце главы делятся на **два типа**, и это разделение явное:

1. **Тематические (strategic practice)** — задачи на конкретный приём текущей главы. Отработка одного навыка.
2. **Смешанные (mixed practice)** — задачи на синтез навыков из нескольких глав. Ближе к экзамену ШАД.

Это **важная структура** именно для ШАД-подготовки. ШАДовские задачи редко на одну тему — обычно на пересечении.

# Структура блока задач

```
## Задачи

### Разминка по теме

Эти задачи — на конкретный приём этой главы. Делайте подряд, набирая навык.

1. ...
2. ...
...

### Смешанные задачи

Эти задачи требуют комбинировать материал нескольких глав.

1. (главы 2, 3) ...
2. (главы 3, 4) ...
```

# Соотношение

| Тип | Количество | Доля |
|---|---|---|
| Тематические | 15-20 | ~70% |
| Смешанные | 5-10 | ~30% |

Точные числа зависят от объёма главы. Общий ориентир — **25-30 задач** на главу.

# Когда применять

- В **каждой главе**, начиная со второй. (В первой смешанных быть не может — материала ещё мало.)

# Связка

`tasks_strategic_vs_mixed` + `tasks_difficulty_levels` — двумерная классификация: по теме (одна/много) и по сложности (разминка/основа/трудная). Это даёт навигацию по блоку задач.

# Маркировка смешанных задач

Каждая смешанная задача должна **в скобках указывать**, материал каких глав требуется. Это даёт читателю выбор: «эту главу прошёл, эту нет — пропускаю».

# Сборщик

Этот паттерн — это **структурное требование** для генератора задач (этап 5 пайплайна). Скрипт-генератор:

1. Получает на вход список глав.
2. Генерирует 15-20 тематических задач (только на материал текущей главы).
3. Генерирует 5-10 смешанных (на текущую + 1-2 предыдущие).
4. Размечает каждую: тематическая или смешанная, какие главы нужны.
