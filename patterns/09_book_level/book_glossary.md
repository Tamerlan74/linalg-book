# Суть

Каждый **новый термин** при первом введении в тексте помечается **специальной разметкой**. По этой разметке потом автоматически собирается **глоссарий** в конце книги — алфавитный список терминов с краткими определениями и ссылками на главу первого появления.

# Разметка в тексте

При первом упоминании термина в книге пишется:

```markdown
**[вектор]{термин: упорядоченный набор чисел; интуитивно — стрелка с длиной и направлением}**
```

Это **жирное** упоминание + **скрытое определение** в фигурных скобках.

В рендере читатель видит просто **вектор**, а скрипт извлекает определение для глоссария.

# Что вносить в глоссарий

| Вносить | Не вносить |
|---|---|
| Главные термины предмета (вектор, матрица, определитель) | Обычные слова из текста |
| Специальные обозначения ($\det M$, $|M|$, $\vec{v}$) | Стандартные ($+$, $\times$, $=$) |
| Авторские именованные модели («композиция инструкций») | Простые аналогии |

# Когда применять

- При **первом** упоминании термина в книге. Не при каждом — иначе глоссарий будет дублироваться.

# Технические требования

- Маркер `**[термин]{определение}**` — единый формат по всей книге.
- **Только при первом упоминании**. В следующих упоминаниях — просто **термин** без скобок.
- Сборщик отслеживает дубли и выдаёт предупреждение, если термин помечен дважды.

# Структура итогового глоссария

```
# Глоссарий

**Вектор** — упорядоченный набор чисел; интуитивно — стрелка с длиной и направлением. Глава 1.

**Композиция инструкций** — последовательное применение двух матриц-преобразований. Глава 4.

**Матрица** — инструкция преобразования пространства; форма записи — таблица чисел. Глава 2.

...
```

Алфавитный порядок, краткое определение, ссылка на главу.

# Связка

`book_glossary` хорошо работает вместе с **index** (предметным указателем) — но index это отдельная сущность, не паттерн стиля. Index делается финальной сборкой автоматически из всех упоминаний терминов.

# Сборщик

Полностью автоматизируется:
1. По всем главам — поиск маркеров `**[X]{Y}**`.
2. Извлечение пар (термин, определение).
3. Сортировка по алфавиту.
4. Добавление номера главы первого упоминания.
5. Вставка в конец книги как раздел «Глоссарий».
